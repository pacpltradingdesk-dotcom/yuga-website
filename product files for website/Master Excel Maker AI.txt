WHAT IT DOES (ABOUT SECTION)
--------------------------------------------------------------------------------

Heading: What Is Master Excel Maker AI?

Master Excel Maker AI is an intelligent data processing platform built for
Indian sales teams and businesses. It takes your scattered, inconsistent
customer lists and transforms them into clean, organized, and AI-powered
Excel files — without any manual effort.

Whether you have 100 contacts or 50,000, the system automatically:
- Reads and understands your files, no matter what column names you use
- Cleans phone numbers, corrects city/state spellings, removes duplicates
- Scores every lead from 0 to 100 so your team knows who to call first
- Exports a professional, multi-sheet Excel file ready for CRM import


--------------------------------------------------------------------------------
KEY FEATURES
--------------------------------------------------------------------------------

Heading: Everything Your Sales Team Needs

1. SMART FILE READING
   Supports .xlsx, .xls, and .csv files. Automatically identifies columns
   like "Mobile No", "Ph.", "Cell", "Contact" — no manual mapping needed.

2. INTELLIGENT DATA CLEANING
   - Phone numbers standardized to E.164 format
   - City spelling corrections (e.g., "Gujrat" → "Gujarat")
   - State auto-filled from city name using a built-in India geography database
   - Duplicate contacts merged intelligently (3-pass: Phone → Email → Name)

3. NATURAL LANGUAGE FILTERING
   Simply type what you need — in English or Hindi:
   "Gujarat ke saare contacts do"
   "Show me Mumbai leads with phone numbers only"
   "Active customers from Pune with email"
   The AI understands and filters instantly.

4. AI LEAD SCORING
   Every contact receives an automatic lead score (0–100):
   🔥 Hot Lead (70–100)   → Call immediately
   📊 Medium (40–69)      → Send catalog or quote
   ❄️ Cold (0–39)         → Add to nurture campaign

   Score is calculated based on data completeness, product interest,
   positive/negative sentiment, GSTIN presence, and more.

5. AI CUSTOMER PROFILING
   Get a natural language summary for each contact:
   "This B2B buyer from Ahmedabad has shown strong purchase intent,
   has a valid GSTIN, and is likely ready to buy."

6. PROFESSIONAL EXCEL OUTPUT
   - 11-sheet workbook: By State, By City, High Value Leads, Junk Contacts,
     CRM Ready, and more
   - Conditional formatting, color-coded lead categories
   - Single flat sheet for direct CRM import (HubSpot, Salesforce)

7. BATCH PROCESSING
   Point the tool at an entire folder of Excel files. It scans, identifies
   contact sheets, processes them all, and merges into one master file.

FAQ
--------------------------------------------------------------------------------

Q: Do I need to format my Excel file a specific way?
A: No. The AI automatically detects column names like "Mobile", "Ph.", "Cell",
   "Customer Name", "Party Name", etc. Just upload as-is.

Q: Does it work with Hindi data?
A: Yes. The system handles Hindi column names and supports Hindi prompts for
   filtering (e.g., "Gujarat ke contacts chahiye").

Q: Is my data safe?
A: All processing happens locally on your machine or server. Data is never
   stored or sent to third parties (except optional AI API calls for scoring).

Q: Do I need an API key?
A: No API key is needed for basic cleaning, deduplication, and Excel export.
   AI lead scoring and profiling require a Gemini or OpenAI API key.

Q: What file formats are supported?
A: .xlsx, .xls, and .csv files are supported.

Q: Can it handle multiple files at once?
A: Yes. Upload multiple files or use batch mode to process an entire folder.
